# 제어문

# 연산자
- A 연산 B
- 결과 : TRUE , FALSE
- `>`       : 크다
- `>=`      : 크거나 같다
- `<`       : 작다
- `<=`      : 작거나 같다
- `==`      : 같다
- `!=`      : 같지 않다


### 논리 연산
- A 연산 B
- `&`   : 두 조건이 모두 만족해야 TRUE
    - AND ("그리고")

- `|`   : 둘 중 하나의 조건만 만족해도 TRUE
    - OR ("또는")

## apply() 함수
- 행렬의 반복되는 연산을 처리하는 함수
- 형식 : apply(행렬, margin, 함수)
- * margin : 1 : 행, 2 : 열
```
    X <- matrix(1:4, 2, 2)      # 1~4까지 2행 2열 행렬 생성

    apply(x, 1, sum)            # 행렬 x의 행의 합계
    apply(x, 2, sum)            # 행렬 x의 열의 합계
    apply(x, 1, max)            # 행렬 x의 행의 최댓값
    apply(x, 1, min)            # 행렬 x의 행의 최솟값


```
