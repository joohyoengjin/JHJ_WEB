# 설치 및 설정
 
 ## pip install matplotlib
 ## import matplotlib.pyploy as plt - 별칭을 활용해서 주로 사용
 ## 라이브러리 사용개념
    * x축과 y축 데이터를 생성 또는 가져와서 정의 (연속적인 또는 끊어지는 데이터)
    * plt.plot(x축 , y축) - plot(선그래프), bar(막대그래프), pie(파이그래프), scatter(산점도)
    * plt.show()
    * plt.xlabel() 축의 라벨을 표시
    * plt.plot(옵션) 범례
    * plt.title(제목)
